\# 🎧 YouTube to MP3 Converter



A full-stack YouTube to MP3 converter built with \*\*Flask\*\*, \*\*yt-dlp\*\*, and \*\*FFmpeg\*\*.



This project supports:

\- Real-time download progress

\- Automatic MP3 naming using YouTube title

\- Auto-delete after download

\- Modern 3D UI with success animation

\- Desktop and cloud-ready deployment



---



\## 🚀 Features



\- 🔥 Real-time progress bar

\- 🎵 MP3 filename = YouTube video title

\- 🧹 Auto-delete downloaded files

\- ⚡ Fast audio extraction

\- 💻 Desktop-ready (EXE supported)

\- ☁️ Cloud deployable (Render)



---



\## 🛠️ Tech Stack



\- Python

\- Flask

\- yt-dlp

\- FFmpeg

\- HTML / CSS / JavaScript
